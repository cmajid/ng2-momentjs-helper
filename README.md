# ng2-momentjs-helper
Angular2 momentjs datatime helper

## Usage 
Install from npm: 
```
npm install ng2-momentjs-helper --save
```
<Enter>
Import static class MomentjsHelper in head
  
```
import { MomentjsHelper } from 'ng2-momentjs-helper/dist';
```
<Enter>
  
Use in codes..
```
let value = MomentjsHelper.GetJalaliNow();
console.log(value);
```
